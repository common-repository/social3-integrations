Social3: Free Website and Social Media Marketing Tools

== Description ==

Social3 Wordpress Plugin integrates with the oh so free marketing tools at social3.io. Juice your site up today and start gathering e-mail addresses and interacting socially easier than ever before

== Installation ==

1. Upload folder s3-integrations.zip to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. Click on the link `Settings`
1. Register and connect plugin to your account on Social3.io
